
# DSA Cancellation Checker

Python script to check DSA available test dates. If it finds one, it will send an email to one or more addresses with the dates and times.

## Usage

*Settings:*  
Copy `.env.example` as `.env`.  
Then enter your details in the new file.

*Run:*  
```python3 check.py```
